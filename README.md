Codecraft Task 1: Caesar Cipher

A simple Python program to encrypt and decrypt text using the Caesar Cipher algorithm.
How It Works
The program asks the user for:
1.	A message to encrypt or decrypt.
2.	A shift number (how many letters to shift the alphabet).
3.	The choice to either "encrypt" or "decrypt" the message.
How to Run
Run the caesar_cipher.py file in your Python environment (like PyCharm) or from your computer's terminal. Follow the on-screen prompts to test the encryption!
Example
* Original text: test
* Shift: 3
* Encrypted text: whvw
